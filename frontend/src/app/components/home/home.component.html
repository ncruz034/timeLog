Notes on Implementation

Work Orders

As of now the order number must be inserted manually; in the production application, this value will be provided automatically.

Other features that will be available in the production application, are the linking of client, and project information to the work order creation process.

Users

The avility to create time through the user menu, although it is implemented, will not be available at this time.

As of this version the application access levels is implemented; however, all users have the same access level in this release.
